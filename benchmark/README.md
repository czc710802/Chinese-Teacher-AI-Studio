# Benchmark Center

This directory stores all P1.5 AI grading quality benchmark data for Chinese Teacher AI Studio.

- `history/`: imported historical essays and old reports.
- `reports/`: per-essay new reports, comparisons, teacher reviews, Word/PDF/Markdown files.
- `result/`: benchmark summary and model comparison output.
- `charts/`: benchmark chart artifacts.
- `export/`: final Benchmark reports and downloadable packages.
- `logs/`: `benchmark.log` and retry diagnostics.
- `config/`: typed config and JSON Schema.

Benchmark runs are independent from production grading, Feishu, NAS, teacher dashboard, and student flows.
