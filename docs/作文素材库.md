# 作文素材库

## 目的

帮助学生积累可核实、可迁移、能服务观点的写作素材，避免“堆名人名言”式使用。

## 素材分类

| 分类 | 示例 |
|---|---|
| 人物事例 | 历史人物、科学家、普通劳动者、时代楷模 |
| 事件与数据 | 社会事件、权威统计、公共议题案例 |
| 名言与经典 | 经典原文、作家观点、政策或报告表述 |
| 文化知识 | 文学、艺术、哲学、传统技艺与典籍 |
| 方法与结构 | 论证方法、开头结尾、过渡、段落模板 |

## 单条素材规范

应包含标题、正文、出处、适用主题、关键词、可论证的观点、使用提醒和核实状态。引用信息不可确认时不得标记为“可直接引用”。

## 使用原则

- 素材必须服务观点，并补充分析，不以罗列替代论证。
- 引用应准确、适量；避免虚构事例和未经核实的名言。
- 同一素材应提供多主题迁移建议，防止机械套用。

## 当前状态

数据库已有 `material_library` 表，包含分类、标题、正文、来源、标签和使用次数字段；学生端和教师端的完整素材库界面待建设。
