# 学生档案设计

## 目标

以多次作文为单位记录学生的能力变化、共性问题和下一步训练建议，支持“批改—练习—再提交—复盘”的闭环。

## 档案内容

| 模块 | 内容 |
|---|---|
| 基本信息 | 姓名、学号、年级、学校、所属班级 |
| 作文历史 | 任务、提交时间、版本、总分、教师批注、报告链接 |
| 能力趋势 | 总分及审题、内容、结构、语言、发展等维度的时间趋势 |
| 问题画像 | 高频问题、对应示例、出现次数、改善状态 |
| 成长亮点 | 优秀句段、稳定优势、进步节点 |
| 训练计划 | 个性化建议、推荐题目/素材、练习完成情况 |

## 数据映射

`student_profiles` 保存趋势、问题、成长报告和建议；`essays`、`ai_reviews`、`teacher_comments` 保存单篇证据；AI 训练相关表保存训练过程。

## 当前状态

学生成长档案页面与学生维度分析接口已实现。训练闭环、目标设定与更细的维度可视化列入后续任务。
