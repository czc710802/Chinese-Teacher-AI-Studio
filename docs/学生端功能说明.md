# 学生端功能说明

## 已实现功能

| 功能 | 说明 |
|---|---|
| 学生登录与首页 | 进入个人学习工作台 |
| 公网访问 | 可通过登录页公网演示入口或 `https://pi.zhenwanyue.icu` 在手机浏览器访问 |
| 班级与任务查看 | 学生由教师在班级管理中添加后，查看待完成作文任务 |
| 作文提交 | 输入作文文字，按任务完成提交 |
| 拍照/图片上传 | 直接上传作文照片或图片，由 AI 自动识别图片文字并批改 |
| 批改结果 | 查看分数、教师总评、自然段详细旁批、文章亮点、问题、段落改写示范与升格建议 |
| 深度建议 | 建议分为问题诊断、逻辑分析、修改步骤、示例方向；结果页不再展示五项“多维分析”栏目 |
| 思维教练 | 展示逻辑思维能力 30 分、思维深度星级、思维提升建议、苏格拉底式追问和深度修改闭环 |
| 原文润色提升对比 | 在原文中内联标出保留原文、新增内容和删除内容：原文黑色、新增蓝色、删除红色 |
| 整篇升格文章 | 按高考高分目标进行深度改写，重构薄弱段落、强化论证和语言表达；学生端导出 Word/PDF 时同步包含整篇升格文章 |
| 成长档案 | 查看历史作文、成绩趋势、思维成长档案和个性化训练建议；思维成长覆盖逻辑能力、思辨能力、论证能力、材料分析能力、语言表达能力和修改能力，并结合已批改作文展示逐篇详细分析 |
| AI 辅导 | 使用对话、仿写、升格、模拟阅卷等学习能力 |

## 使用流程

登录 → 查看任务 → 选择文字提交或拍照/图片上传 → AI 自动识别并批改 → 阅读批改和思维教练追问 → 完成深度修改任务 → 在下一次作文中复盘。

## 使用提醒

拍照或上传图片时应保证画面清晰、完整、光线充足。AI 意见用于学习参考；评分争议和最终评价以教师意见为准。
