# 教师端功能说明

## 已实现功能

| 功能 | 说明 |
|---|---|
| 教师登录与首页 | 使用教师角色进入教学工作台 |
| 公网演示入口 | 登录页和教师首页显示公网演示入口，可复制 `https://pi.zhenwanyue.icu` 或直接打开，用于手机端访问、课堂展示和线上演示 |
| 修改密码 | 教师可在教师首页输入当前密码并设置新密码 |
| 班级管理 | 创建班级、查看班级学生、管理成员 |
| 作文任务发布 | 设置题目、题干、作文类型、满分和截止时间 |
| 班级作业 | 按班级和发布时间汇总已发布作文任务，显示已提交、未提交、已批改篇数，支持前往批改和任务作文导出；导出内容包含学生端批改结果里的整篇升格文 |
| 发布任务管理 | 按发布时间展示已发布任务；完全相同的重复任务只保留一条，优先保留已有学生提交的任务；发布按钮会在提交中禁用，避免重复点击生成重复任务 |
| 批改记录 | 汇总所有已批改作文，支持搜索、按班级筛选、批量导出和点击查看单篇批改结果；单篇与批量导出均包含 AI 批改后的整篇升格文章 |
| 班级作文查看 | 查看学生提交情况、作文原文和 AI 批改结果 |
| 教师二次批注 | 对单篇作文补充文字评论和分数调整 |
| 数据分析 | 查看班级作文统计、学生成长信息和班级思维分析 |
| 班级思维分析 | 统计全班最薄弱能力、逻辑思维能力均值和教学建议，例如不会分析原因、不会组织完整论证链、不会让材料证明观点 |
| 报告导出 | 导出单篇作文、班级作业作文、已批改作文汇总、学生档案、班级分析的 Word/PDF；单篇作文、班级作业和批改记录导出会带出 AI 批改后的整篇升格文章 |

## 使用流程

打开公网演示入口 → 创建班级 → 发布任务 → 在“班级作业”查看提交/未提交/已批改统计 → 前往批改 → 在“批改记录”查看或导出已批改作文 → 分析班级共性问题和思维薄弱项 → 导出或布置后续训练。
