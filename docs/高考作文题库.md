# 高考作文题库

## 目的

沉淀可按地区、年份、主题和训练目标检索的作文题目，为教师发布任务和学生专项训练提供来源。

## 题目数据设计

每一道题目建议包含：

| 字段 | 说明 |
|---|---|
| `title` | 题目名称或简短标题 |
| `year` / `region` | 年份和命题地区；原创题标记为“校本” |
| `type` | 材料、命题、话题、漫画、任务驱动等 |
| `prompt` | 完整题干与材料 |
| `requirements` | 字数、文体、限制条件 |
| `themes` | 主题标签，如成长、青年、科技、文化 |
| `difficulty` | 基础、进阶、挑战 |
| `skills` | 审题、立意、论证、叙事、语言等训练点 |
| `reference` | 命题来源、授权/版权信息 |

## 分类建议

- 价值与成长：责任、理想、选择、规则、青春。
- 社会与时代：科技、乡村、劳动、共同体、媒介。
- 文化与传承：经典、传统、创新、文明互鉴。
- 思辨关系：快与慢、得与失、个人与群体、守正与创新。
